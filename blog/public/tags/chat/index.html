Chủ đề 9: Ứng dụng thực tế - Xây dựng Chat System

1. Giới thiệu dự án Chúng ta sẽ xây dựng một phòng chat đơn giản nơi mọi người có thể vào và nhắn tin cho nhau. Tính năng: Chat Real-time, hiển thị danh sách người online. Công nghệ: WebSocket. 2. Kiến trúc hệ thống Backend: Đóng vai trò WebSocket Server. Lựa chọn 1: Node.js + Socket.io (Dễ triển khai, code ngắn). Lựa chọn 2: Spring Boot + WebSocket (Robust, kiểu dữ liệu chặt chẽ). Frontend: HTML/CSS/JS thuần. Kết nối tới Backend. 3. Triển khai Backend (Node.js) const io = require('socket.io')(3000, { cors: { origin: "*" } }); io.on('connection', socket => { // 1. Nhận tin từ Client A socket.on('send-chat', message => { // 2. Phát lại cho B, C, D... io.emit('receive-chat', { user: socket.id, text: message, time: new Date().toLocaleTimeString() }); }); }); 4. Triển khai Frontend <!-- HTML đơn giản --> <div id="messages"></div> <input type="text" id="msgInput"> <button onclick="send()">Gửi</button> <!-- JavaScript xử lý --> <script src="https://cdn.socket.io/4.0.0/socket.io.min.js"></script> <script> const socket = io('http://localhost:3000'); // Gửi tin function send() { const text = document.getElementById('msgInput').value; socket.emit('send-chat', text); } // Nhận tin socket.on('receive-chat', data => { const div = document.createElement('div'); div.innerHTML = `<b>${data.user}</b>: ${data.text} <i>(${data.time})</i>`; document.getElementById('messages').appendChild(div); }); </script> 5.
