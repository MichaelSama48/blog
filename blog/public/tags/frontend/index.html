Làm Chủ DOM: Biến Website Trở Nên Sống Động

1. DOM là gì? DOM (Document Object Model) là mô hình dạng cây biểu diễn toàn bộ trang HTML của bạn. Trình duyệt tạo ra DOM để JavaScript có thể can thiệp vào. Ví dụ: Thẻ <body> là cha của <div>, <div> là cha của <h1>. 2. Truy xuất phần tử (Selectors) Để thao tác, bước 1 là phải “túm” được phần tử đó. // 1. Theo ID (Chỉ lấy 1 cái) const title = document.getElementById("header-title"); // 2. Theo Class (Lấy danh sách) const buttons = document.getElementsByClassName("btn"); // 3. QuerySelector (Mạnh mẽ nhất - Lấy phần tử dầu tiên tìm thấy) const mainDiv = document.querySelector(".container #main"); // 4. QuerySelectorAll (Lấy tất cả) const allLinks = document.querySelectorAll("a.nav-link"); 3. Thao tác nội dung và Style const box = document.querySelector(".box"); // Đổi nội dung text box.innerText = "Xin chào Javascript!"; // Đổi HTML bên trong box.innerHTML = "<strong>In đậm nè</strong>"; // Đổi Style (CSS) box.style.backgroundColor = "tomato"; box.style.fontSize = "20px"; box.classList.add("active"); // Thêm class CSS 4.
