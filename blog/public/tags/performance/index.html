Kiến trúc JVM (Java Virtual Machine) Khi bạn chạy lệnh java HelloWorld, điều gì sẽ xảy ra? Classloader: Tải file .class (bytecode) vào bộ nhớ. Bytecode Verifier: Kiểm tra xem code có an toàn và hợp lệ không (tránh tràn bộ nhớ, truy cập trái phép). Interpreter: Dịch bytecode sang mã máy (machine code) từng dòng một. JIT Compiler (Just-In-Time): Đây là vũ khí bí mật của Java. Nó phát hiện các đoạn code được chạy nhiều lần (“hot spots”) và biên dịch chúng thành mã máy thuần túy để chạy cực nhanh, gần bằng C++. 1.2. Write Once, Run Anywhere?
