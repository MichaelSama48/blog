JavaScript Under The Hood: V8 Engine, Call Stack và Hoisting

1. JavaScript Engine hoạt động thế nào? Code JS bạn viết chỉ là văn bản. Máy tính không hiểu. Cần một “Bộ máy” (Engine) để dịch nó. Nổi tiếng nhất là V8 Engine (của Chrome và Node.js). Quy trình của V8: Parser: Phân tích cú pháp code thành AST (Abstract Syntax Tree). Ignition (Interpreter): Dịch AST sang Bytecode và chạy ngay lập tức (giúp khởi động nhanh). TurboFan (Compiler): Trong lúc code chạy, nó tối ưu hóa các đoạn code lặp lại thành mã máy (Machine Code) để chạy siêu nhanh. 2. Call Stack & Memory Heap Memory Heap (Bộ nhớ) Nơi lưu trữ các biến, object, hàm. Nó là một kho chứa hỗn độn.
